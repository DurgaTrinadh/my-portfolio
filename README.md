# 🌐 My Portfolio — Ranganadham Durga Trinadh

A personal portfolio website built to showcase my projects, skills, and internship experience as a CSE student and full-stack developer.

🔗 **Live Site:** [my-portfolio-roan-theta-10.vercel.app](https://my-portfolio-roan-theta-10.vercel.app/)

---

## ✨ Features

- **Animated Hero Section** — Dynamic intro with smooth scroll and floating elements
- **About Section** — Education, internship highlights, and a downloadable resume
- **Skills Section** — Frontend, programming languages, and CS tools
- **Projects Section** — Filterable project cards (All / Full-Stack / Frontend)
- **Testimonials** — Kind words from peers and mentors
- **Contact Form** — Email-integrated contact form with live feedback
- **Fully Responsive** — Works across desktop, tablet, and mobile

## 🚀 Getting Started

### Prerequisites

- Node.js v18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/DurgaTrinadh/my-portfolio.git
cd my-portfolio
```

### Run the Frontend

```bash
cd frontend
npm install
npm start
```

### Run the Backend

```bash
cd backend
npm install
node index.js
```

The frontend runs on `http://localhost:3000` and the backend on `http://localhost:5000` by default.

---

## 🌍 Deployment

- **Frontend** deployed on [Vercel](https://vercel.com/)
- **Backend** deployed on [Render](https://render.com/)

To deploy your own version, fork the repo and connect it to Vercel (frontend) and Render (backend). Set any required environment variables (e.g. email credentials for the contact form) in your hosting dashboard.
